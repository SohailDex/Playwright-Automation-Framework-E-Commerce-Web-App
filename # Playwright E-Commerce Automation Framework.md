# Playwright E-Commerce Automation Framework  
  
## Overview  
End-to-end automation framework built using Playwright and TypeScript  
to automate an e-commerce web application.  
  
## Tech Stack  
- Playwright  
- TypeScript  
- Page Object Model  
- GitHub Actions CI  
  
## Test Coverage  
- Login validation  
- Product add to cart  
- Checkout flow  
  
## Run Tests  
npm install    
npm test    
  
## Reports  
npx playwright show-report
